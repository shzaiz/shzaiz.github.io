关于这个博客(或者说 Wiki)

其实是仿照jyywiki做的一个小东西, 主要是因为长期以来的不良 Markdown 风格让我用 hexo 之类的博客工具很麻烦(渲染炸掉/根本没办法生成博客), 因此就希望造一个"格式不会炸的太严重", "自己能掌握几乎全部的细节", "可以自由的删改 markdown parser"的Wiki.

note里面是一些小笔记, slide是一些演示文稿方便演示文件(方便自己做演示嗯), 杂项里面是一些奇怪的东西, 有一部分是以前写过的讲稿(高中数学入门, 今年暑假当家教写的); 还有一部分完全是乱写.

所以这个 Wiki 应该是处于最基本的内容展示的目的, 让我基本上可以毫不费心的从我的笔记 repo 转到网页方便浏览的一个过程. 后来的话看着完善吧, 不要抱太大期望, 本来就是我兴趣使然做的一个小东西哈哈哈哈.

这个 Wiki 生成的 HTML 完全由一段小的 python 代码实现的, 具体地, 我只需要丢进去一个满满的 markdown 文件夹, 它就可以自动给我转化成像这样的 HTML. 感觉也是挺好玩的一件事情.
