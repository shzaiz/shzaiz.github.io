最短路

Overview

  • 单源最短路
    • 所有边权正数
      • 朴素Dij \((n^2)\)
      • 堆优化Dij( \(n\log m\) )
    • 存在负权边
      • Bellman-Ford O(nm)
      • SPFA 一般O(m), 最坏O(nm)
  • 多源汇最短路: O( \(n^3\) )

稀疏与稠密

  • 如果 \(n^2 << m\) , 那么是稀疏的
  • 如果 \(n^2 \sim m\) , 那么是稠密的

朴素版的Dij ::: section

思路: 有集合S, 当前已经确定是最短距离的点

(1) 初始化距离, 让dist[1]=0, dis[i] = INF

(2) for(i=0..n), 找到不在S中的距离最近的点, 比如称作T, 加入S中

(3) 用t更新其他点的距离 (dis[s]>?dis[t]+w)

Code. (Buggy)

#include <cstring>
#include <iostream>
#include <algorithm>
int n, m; 
using namespace std;
const int N=510;
#define INF 0x3f3f3f

int g[N][N];
int dist[N];

bool st[N];

int dij(){
  memset(dist, 0x3f, sizeof dist);
  for(int i=0; i<n; i++){
    int t = -1;
    for(int j=i; j<=n; j++){
      if(!st[j] && (t==-1 || dist[t]>dist[j])){
        t = j;
      }
    }
    st[t] = 1;
    for(int j=1; j<=n; j++) 
      dist[j] = min(dist[j], dist[t]+g[t][j]);
  }
  if(dist[n]==0x3f3f3f) return -1;
  else return dist[n];
}

int main(){
  cin>>n>>m;
  for(int i=1; i<=n; i++){
    for(int j = 1; j<=n; j++){
      if(i==j) g[i][j] = 0;
      else g[i][j] = INF;
    }
  }
  while(m--){
    int a,b,c; cin>>a>>b>>c;
    g[a][b] = min(g[a][b],c);
  }
  int t = dij();
  cout<<t;
  return 0;
}
