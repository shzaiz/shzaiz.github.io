排序问题

快速排序

  1. 基本思想: 分治法.

  2. 过程描述

(1)确定元素: 首先随机选取一个数$x$;

(2)调整数组: 把小于$x$的放在这个位置的左边, 大于$x$的放在这个位置的右侧.

(3)递归执行: 一直这样递归下去.

  1. 问题: 如何进行(2)操作

(i) 开两个额外的数组, $a[],b[]$, 如果当前的数小于等于$x$, 就插入$a$, 否则, 插入$b$, 最后把$a[],b[]$放入原先的数. -- 简单但是不够优美.

(ii) 用两个指针$i$,$j$分别向中间走, 如果$i<x$, 向后移动, 直到$i>x$, $i$停下, $j$也一样. 直到i,j都错位了, 现在把两个指针的内容交换一下就行了. 继续往中间走就行了. 正确性证明: 在任何时候, i左边所有的数$\leq x$(算法的语句保证), j右边的数也是大于$x$的. 于是就可以分成相应的区间.

  1. 细节: 边界问题的处理 -- 模板代码

(1) 代码的书写

void qst(int a[], int l, int r){
    if(l>=r) return;
    //(1)
    int x = a[l]; //Alterate r, (l+r)/2
    int i = l-1, j=r+1; //Move first, then compare and swap
    while(i<j){
        do i++; while(a[i]<x);
        do j--; while(a[j]>x);
        if(i<j) swap(a[i],a[j]);
    }
    qst(a, l, j); // If x:=r, then this line should be qst(a,l,i-1)
    qst(a,j+1,r); // and this line should be qst(a,i,r)
}

归并排序

  1. 基本思想: 分治.

  2. 过程描述:

(1) 确定数组中心为分界点$(l+r)/2$.

(2) 递归排序左边和右边;

(3) 把左边和右边两个有序的数组合二为一.

  1. 问题: 如何合二为一

假设有a, b, res数组, 开两个指针i,j指向a, b的头. 比较a[i]与b[j]中较小的那一个, 放到答案res数组中, 较小的那一个指针向后移动一位. (相同的时候需要把第一个移动到答案中, 这样做是稳定的(Def. 相同值的会在相同的前后的顺序即可.) )

正确性证明: 因为取的是两个数组中的较小值, 每次总是能够按照从小到大的选取.

  1. 模板代码
// We need a tmp array to store relations...
void mgsort(int a[], int tmp[], int l, int r){
    if(l>=r) return ;
    //(1)
    int mid = (l+r)>>1; // Bracket missing is also okay.
    mgsort(a, tmp, l, mid);
    mgsort(a, tmp, mid+1, r);
    int k = 0, i = l, j = mid+1;
    // merge
    while(i<=mid && j<=r){
        if(a[i]<=a[j]){
            tmp[k++] = a[i++];
        }else{
            tmp[k++] = a[j++];
        }
    }
    // Is there anything missing in the loop?
    while(i<=mid) tmp[k++] = a[i++];
    while(j<=r) tmp[k++] = a[j++];
    for(i=l,j=0; i<=r; i++, j++) a[i] = tmp[j];
}

整数二分

目标: 找到一个性质的边界

如果要找红色的点:

  • 找一个中间值mid=(l+r+1)>>1
  • 判断中间值是不是满足性质check(mid)
    • 如果是true, 那么l=mid
    • 如果是false, r=mid-1
  • 循环到第一步, 直到r>=l.

如果要找上图绿颜色的分界点:

  • 找一个中间值mid=l+r>>1
  • 判断中间值是不是满足性质check(mid)
    • 如果是true, r=mid
    • 如果是false, l=mid+1
  • 循环到第一步, 直到r>=l

vint div(vint &a, int b,int &r){

双指针

  1. 通用模板
for(i = 0, j=0; i<n; i++){
    while(j<i && check(i,j)) j++;
    // 问题的具体逻辑

核心思想: 如果我们使用F(i,1,n)F(j,1,n), 那么有$n^2$种组合. 但是这样我们就可以做$n$的复杂度. 两个指针的移动不超过2n.

基本例2. 最长连续不重复子序列(ACW799)

题目大意: 给定一个长度为 n 的整数序列，请找出最长的不包含重复的数的连续区间，输出它的长度。

暴力做法: 枚举起点,终点, 判断j到i是不是成立.

---------------
    ^      ^ 
    Green  Red

优化: 每次红颜色的指针往后移动一个, 绿颜色的指针也会往后移动一位. (证明: 如果对应当前红颜色的指针绿颜色往前移动了, 一个没有重复子区间的内部也没有重复子区间, 矛盾! )

#include <bits/stdc++.h>
using namespace std;
#define MAXN 100010
int n, a[MAXN];
int res = -1;
int s[MAXN]; // [j..i]每一个数出现的次数
bool check(){
    
}
int main(){
    scanf("%d", &n);
    for(int i=1; i<=n; i++) scanf("%d", &a[i]);
    for(int i=1, j=1; i<=n; i++){
        s[a[i]]++;
        while(j<=i && s[a[i]]>1) {
            s[a[j]]--; j++;
        }
        res = max(res, i-j+1);
    }
    printf("%d", res);
}

基本例. ACW802

#include <bits/stdc++.h>
using namespace std;

using namespace std;
#define MAXN 300007
#define PII pair<int, int>
int n, m;
int a[MAXN], s[MAXN];
vector<int > alls;
vector<PII > add, query;
int find(int x){
  int l=0, r = alls.size()-1;
  while(l<r){
    int mid = l+r>>1;
    if(alls[mid]>=x) r=mid;
    else l = mid+1;
  }
  return r+1;
  
}
int main(){
  cin>>n>>m;
  for(int i=0; i<n; i++){
    int x, c;
    cin>>x>>c;
    add.push_back({x,c});
    alls.push_back(x);
    
  }
  for(int i=0; i<m; i++){
    int l, r;
    cin>>l>>r;
    query.push_back({l,r});
    
    alls.push_back(l);
    alls.push_back(r);
    
  }
  
  sort(alls.begin(), alls.end());
  alls.erase(unique(alls.begin(), alls.end()), alls.end());
  
  for(auto item : add){
    int x = find(item.first);
    a[x]+=item.second;
  }
  
  // prefix
  
  for(int i=1; i<=alls.size(); i++){
    s[i] = s[i-1]+a[i];
  }
  
  //query
  
  for(auto item : query){
    int x = find(item.first), y = find(item.second);
    cout<<s[y]-s[x-1]<<endl;
    
  }
  
  
}

区间合并

给定许多区间, 如果有交集, 就把它合并成一个区间. 输入n个区间, 把所有有交集的合并, 输出合并之后的区间个数.

规定: 只有边界相邻, 仍然合并.

1 2 3 4 5 6 7 8 9
|-|
  |---| |-| |---|
             |--|
合并之后: 
|-----| |-| |---|

(1) 按照区间的左端点排序 (2) 扫描整个区间, 把可以的合并. 维护当前区间start, end; 如果当前第i个区间, 有包含(A), 有交集(B), 外部(C)的关系.

A. 更新完之后不变. B. ed = i.end C. ans ++; st = i.start, ed = i.end

#include <bits/stdc++.h>
using namespace std;
#define MAXN 1000010
#define F(i, j, k) for(int i=j; i<=k; i++)
int n; vector<pair<int, int> > a;
int main(){
  cin>>n;
  F(i, 1, n) {
    int x, y; cin>>x>>y;
    a.push_back({x,y});
  }
  sort(a.begin(), a.end());
  int cl=a[0].first, cr=a[0].second, ans=1;
  for(int i=1; i<n;i++){
    auto cur = a[i];
    if(cur.first > cr){
      cl = cur.first;
      cr = cur.second;
      ans++;
    }else if(cur.second>cr){
      cr = cur.second;
    }
  }
  cout<<ans<<endl;
  
}
