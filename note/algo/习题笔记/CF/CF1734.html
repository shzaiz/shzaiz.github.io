A. 枚举

枚举三个数 \((a_i, a_j, a_k)\) , \(a_i \leq a_j\leq a_k\) , 使得 \(a_i\) 和 \(a_k\) 都变成 \(a_j\) 最小的长度即可.

B. 构造

C. 贪心

有点那个欧拉筛, 能去掉就去掉, 否则就到下一轮, 每一个都来一次.

注意开long long.

char s[MAXN];
bool vis[MAXN];
void solve(){
    int n; cin>>n;
    F(i, 1, n) vis[i] = 0;
    scanf("%s", s+1);
    int ans = 0;
    F(i, 1, n){
        for(int j = 1; j*i<=n; j++){
            int ck = j*i;
            if(s[ck] == '0'){
                if(!vis[ck]) ans += i;
                vis[ck] = 1; 
            }else{
                break;
            }
        }
    }
    cout<<ans<<endl;
}
