双指针

  1. 通用模板
for(i = 0, j=0; i<n; i++){
    while(j<i && check(i,j)) j++;
    // 问题的具体逻辑
}

核心思想: 如果我们使用F(i,1,n)F(j,1,n), 那么有 \(n^2\) 种组合. 但是这样我们就可以做 \(n\) 的复杂度. 两个指针的移动不超过2n.

基本例1. 输出单词

要求: 输入一个单词, 有一个空格隔开, 每行一个单词输出.

#include <bits/stdc++.h>
using namespace std;

int main(){
	char str[1000];
	gets(str);
	int n = strlen(str);
	for(int i=0; i<n;i++){
		int j = i; 
		while(j<n && str[j]!=' '){
			j++;
		}
        // Particular logic
		for(int k=i; k<j;k++){
			cout<<str[k];
		}
		cout<<endl;
		i=j;
	}
}

基本例2. 最长连续不重复子序列(ACW799)

暴力做法: 枚举起点,终点, 判断j到i是不是成立.

---------------
    ^      ^ 
    Green  Red

优化: 每次红颜色的指针往后移动一个, 绿颜色的指针也会往后移动一位. (证明: 如果对应当前红颜色的指针绿颜色往前移动了, 一个没有重复子区间的内部也没有重复子区间, 矛盾! )

#include <bits/stdc++.h>
using namespace std;
#define MAXN 100010
int n, a[MAXN];
int res = -1;
int s[MAXN]; // [j..i]每一个数出现的次数
bool check(){
    
}
int main(){
    scanf("%d", &n);
    for(int i=1; i<=n; i++) scanf("%d", &a[i]);
    for(int i=1, j=1; i<=n; i++){
        s[a[i]]++;
        while(j<=i && s[a[i]]>1) {
            s[a[j]]--; j++;
        }
        res = max(res, i-j+1);
    }
    printf("%d", res);
}

注意不要每一次更新的时候都更新j, 因为没有必要.

离散化

  1. 本质: 把一个大区间中明显不足这么大的区间的数映射到小区间里面.

如:

1  3   100   20000   50000000
          |||  f(x)
           v
0  1    2     3          4

  1. 一些小问题:

    • a[]可能有重复的元素 (需要去重)
    • 如何算出a里面的值映射的结果 - 保证顺序 a->下标 (二分)

  2. 常见写法:

vector<int> alls; // 存储等待离散化的值
sort(alls.begin(), alls.end()); // 排序
alls.erase(unique(alls.begin(), alls.end()), alls.end()); // 去重
// unique是把所有的重复元素去重, 返回数组末尾的端点
// a:     |-------------|
// a-uniq |--------|~~~~|
//                 ^删除
