研究一个问题

发表于 2020-11-21 | 分类于 杂谈

Desc

在OI里, 这是Polya计数的模板题.

长度为\(n\)的环状DNA碱基排序的所有可能情况. 考虑对称.

Preclude

hlf问的. 感觉是polya定理.

Trans

考虑所有的对称, 旋转, 反转的情况.

考虑旋转的时候不动点\(f\)的个数, 也就是 \[ f = \sum 4^{\gcd(n,i)}. \] 对于反转, 分类讨论, 如果\(n\)为奇数, \(n-1\over 2\)个循环节是2, 1个循环节是1. 此时的ans为 \[ a=n\cdot 4^{n-1\over 2} \] 对于\(n\)为偶数:

  • n/2个，切开两个环之间的内容. 同上.
  • n/2个，把环分成两半，n/2个循环节.

\[ a={n\over 2}(4^{n+2\over2}+4^{n\over 2}) \]

最终答案: \[ {a+b}\over 2n \]

Polya:

Permutation group

Theory fundamentals

  • Permutation group

    • A mapping for the original elements and current elements

    • if 1 -> a1 , 2 -> a2 , ..., n -> an write as

    • \[ \left(\begin{array}* 1&2&\cdots&n\\ a_1&a_2&\cdots&a_n \end{array} \right) \]

    • Full mapping therefore it is inversable.

  • Multiplication

    • Means composing two different perms.
    • eg.
      • Let f be {156243}, let g be {624351}.
      • Consider fg
      • Obviously fg = {651234}

  • Representing by perm cycle

    • eg. (1 4 3) means 1 --> 4 4-->3 3-->1.

    • we may broke down a common group to a cycle mode

    • eg.

      • \[ \left(\begin{array}* 1&2&3&4&5\\ 3&5&1&4&2 \end{array} \right) \]

      • can be described as (1 3)(4 5)(2)

    • (Semi-)Prove that every group like this can break down like this

      • Regard each elem as a node
      • if a --> b , then add an edge from a to b
      • Obviously every node's indeg and outdeg equals 1.
      • It has to be a directed cycle.

    • Demostrate that collection at the view of graph

    • The len of the previous one (1 3)(4 5)(2) is 3.

  • Equivalent counting problem

    • eg.

      • Color Black and White on 2*2 square.
      • If we fix the corner, the answer will be \(2^4=16.\)
      • If we define a rotating action which rot(0) == rot(90) == rot(180) == rot(270)
      • The answer will be 6 left.

    • Characteristic of these problems

      • Reflexivity: If a=b, b=a
      • Transitivity: if a=b, b=c, a=c.

    • Solving this problem

      • let F be the Replacement Group
      • if a in the RG, b in the RG, then ab in RG.
        • It's obvious. for that is the def.

    • Burnside's lemma

      • Fixed point

        • To a transformation when before and after transform that
        • the matrix is the same
        • let C(f) be the sum of the number.

      • Total Equivalent sum equals \(\frac{1}{n(C_i)}\Sigma_{n(C_i)} {C(i)}\).

        let n(ci) be the count of ci

      • Get each C(f)

        • Mark the grids \[ \begin{array} *1&2\\4&3 \end{array} \]

        • Consider every condition that are Equivalent :

          • Theory
            • Break it down to perm cycle mode.
            • Compute with Grade 2 knowledge.
            • get C.
          • Example
            • Rotate 180 deg conterclockwize means (1 3) (2 4)
            • According to Grade 2 knowledge, obviously there are \(2 \times 2 = 4\) Cs
            • continue getting Cs until all of which was taken.
            • Use \(\frac{1}{n(C_i)}\Sigma_n {C(i)}\) to get total conditions

        • Generalize

          • If Replacement can be broke down to perm cycle and its len is n,
          • Obviously Each len has the same color therefore:
          • If we have n colors ready to dye,then the C(n) equals
          • \(C(f) = k^{m(f)}\).

    • Pólya's lemma

      • The total sum of the equivalent group equals:
      • Mean of the \(k^{m(f)}\) in Replacement Group f.

Analysis

There are 2 operations in total.

  • Rotate

    • If rotate across i circles: i,2i, would be inside a loop...

    • So obviously the len of the cycle would be \(n/\gcd(i,n)\).

    • There are \(\gcd (i,n)\) loops

    • so the total sum of fixed points are:

    • \[ a = \sum^{n-1}_{i=0}t^{\gcd(i,n)} \]

  • Flip

    • Note there are two subproblems
    • If n is odd:
      • n SA (Symmetry axis)
      • (n+1)/2 cycles
      • \(b = nt^{(n+1) / 2}\)
    • If n is even:
      • Obviously \(b=\frac{n}{2}(t^{n/2+1}+t^{n/2})\)
