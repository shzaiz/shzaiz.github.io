数列常用解题方法

题型. 求数列的通项

  • 累加法
    • 特征: 两边系数相同, 可加.满足\(a_{n+1}-a_n = f{(n)}\). 满足\(f(n)\)可以求和.
  • 待定系数法
    • 特征: 两边是多项式的内容.
    • 方法: 设左边的等于右边的另一个系数. 例如\(F(n)=kF(n-1)\).
    • 注意: 两边\(n\)变成\(n-1\)的时候需要对于所有的\(n\)进行变换.
  • 指数式的处理
    • 方法: 对于\(a_{n+1}=3a_n+2^n\). 两边同时除以\(2^{n+1}\). 注意次方需要看左边的下标系数.
  • 特征根法
    • 令\(a_{n+1} = \frac{pa_n+q}{ra_n+s}\), 特征根\(x = \frac{px+q}{rx+t}\) 两边同时减去任意一个根, 然后化简就行.
  • 二阶常系数递推的处理
    • 假设\(a_{n+2}+Aa_{n+1} = B(n_{a+1}+Aa_n)\). 然后对应系数相等即可.
  • 对于高次(\(x^m\),m是一个常数)的处理方法.
    • 取出同底的对数. 然后对于这个进行拆项.
  • 奇技淫巧
    • 迭代法: 如果感觉看上去比较好凑的话, 可以试一试进行迭代玩一玩.
    • 配凑法: 从系数, 次数进行比较就行.

题型. 关于\(S_n\)的常用方法

  • 求解通项

    • \[ a_n= \begin{cases} S1\qquad \qquad\quad n=1\\S_n-S_{n-1} \qquad n\geq 2\end{cases} \]

  • 通项的最值问题

    • 等差数列: \(S_n= \frac{d}{2}n^2+(a_1-\frac{d}{2})n\). 从而进行使用二次函数最值.

题型. 数列的求和

  • 等差乘等比的处理方法

    • 假设\(a_n = b_nc_n\).那么\(\sum_{i=1}^{n}a_n = \sum_{i=1}^{n}b_ic_i\).

    • 所以 \(q\sum_{i=1}^{n}a_n = q\sum_{i=2}^{n+1}b_ic_{i+1}\).

    • 两式相减, 及可以得到

    • \[ (1-q)\sum_{i=1}^na_i = b_1c_1 + q\sum_{i=2}^{n}b_ic_i+b_{n+1}c_{n+1} \]

    • 根据等比数列求和公式求和就行了.

  • 分数裂项

    • 情形1. 假设\(\{a_n\}\)是两个等比数列. 观察\(k\frac{na_n-a_{n-1}}{a_ia_{i-1}}\).
    • 情形2. \(\frac{1}{\sqrt {a_i}+\sqrt{a_{i+1}}}\). 进行分母有理化.

题型. 数列中不等式的相关证明

  • 放缩法
    • 裂项
      • 可以魔改里面的东西, 让他们稍微大一点(看不等式的方向). 然后
      • 微调(放一项再搞).
      • 换方法(差远了)
    • FAKE等比放缩:
      • 依据等比数列求和公式\(\lim_{n\to \infin}S_n = \frac{a_1}{1-q}\).
        • 如果出问题, 往后移动一项再搞.
      • 依据\(q\)的性质搞.
        • \(\frac{a_n}{a_n-1}<q\)进行构造公比和\(a_1\).
