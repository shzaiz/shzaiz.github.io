Desc

给出一个 n 行 m 列的矩阵，n*m 个点每个点都站着一个单 位。凯特琳将使用【和平使者】r+c 次，技能效果：对某行/列 中所有单位造成 1 单位伤害。对行总共使用 r 次，对列总共使 用 c 次。黑默丁格统计出有 s 个点受到了奇数单位点伤害，他想知道 凯特琳有多少种方案做到这样的效果。两种方案不同当且仅当存在某行/某列被使用技能的次数不同，最后的答案需要 mod 1000000007。

Input

五个整数 n，m，r，c，s 如题。

Output

一个整数表示答案如题。

Sample Input

2 2 2 2 4

Sample Output

4

约束

所有输入数据都是整数。 由于如果随机出大数据的话基本答案都是 0，所以我保留了三个点使得其答案为 x。我可以以题答的形式提供给你们一个答案为 x 的输入数据。

Idea

想到枚举奇多少行奇数, 多少列奇数, 那么奇数点的就可以算出来了. 如果有\(a\)个奇数行, \(b\)个奇数列, 那么一共有\(n=a(m-b)+b(n-a)\). 我们可以枚举\(a\),那么\(b\) 为

\(\displaystyle b={(n-am)\over(b-2a)}\) .对于每对可行的\((a,b)\),a 有 C\((n,a)\)种放法,b 有 C\((m,b)\)种放法.
